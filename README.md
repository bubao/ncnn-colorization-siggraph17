# ncnn-colorization-sigagraph17
Example of colorization with ncnn SIGGRAPH 2017.

## Prerequisites<br>
* Linux or Windows<br>
* Prebuilded NCNN lib with headers<br>
* Prebuilded OpenCV library with headers
* CPU or NVIDIA GPU + Vulkan<br>

## Usage:
```
COLOR_GAN -i 17.jpg
```
